# Security Policy

## Supported Versions

Security fixes are provided for the latest `main` branch.

| Version | Supported |
| ------- | --------- |
| main    | yes       |
| others  | no        |

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

- Email: `opentigerdev@gmail.com`
- Fallback: open a private security advisory in this repository

Include the following information:

- A clear description of the issue
- Reproduction steps or proof of concept
- Potential impact
- Suggested mitigation (if available)

### Response targets

- Initial acknowledgment: within 72 hours
- Triage decision: within 7 business days
- Status updates: at least every 7 business days while active

If the report is accepted, we will coordinate disclosure timing with the reporter.
